# Integrator SOP Architect - Simple Version

A simplified version of the SOP Architect that runs on Vercel free tier.

## Features
- SOP generation from process descriptions
- Audio transcription
- Guardrail library
- Markdown export

## Deployment

Deploy to Vercel:
1. Push to GitHub
2. Go to https://vercel.com/new
3. Import the repository
4. Deploy

## Local Development

npm install
npm run dev
